# 📒 Address Book Project

## 📌 Project Overview
The Address Book project is a console-based application developed using the C programming language. This project helps users store and manage contact information such as names, phone numbers, email addresses, and addresses in an organized manner.

---

## 🚀 Features
- Add new contacts
- View saved contacts
- Search contacts by name or phone number
- Edit existing contact details
- Delete contact records
- File handling for permanent data storage

---

## 🛠️ Technologies Used
- C Programming
- Data Structures
- File Handling
- VS Code / GCC Compiler

---

## 💡 Project Description
The Address Book project is designed to simplify contact management through a user-friendly console interface. Users can perform operations like adding, updating, searching, and deleting contact information efficiently. The application uses file handling techniques to store data permanently, ensuring records are maintained even after the program is closed.

This project helped improve knowledge in:
- Structures in C
- Functions
- File Operations
- Data Management
- Problem Solving

---

## 🎯 Objective
The main objective of this project is to develop a simple and efficient digital contact management system while gaining practical experience in C programming and real-time application development.

---

## ▶️ How to Run
1. Clone the repository
2. Open the project in VS Code
3. Compile the program using GCC compiler
4. Run the executable file

```bash
gcc address_book.c -o addressbook
./addressbook
```
